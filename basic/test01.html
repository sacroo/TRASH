#p 태그 : 문단 태그, 시작과 끝 태그가 나위어져 있음. p태그를 사용하면 문단의 안 글자는 가로크기 한줄 가득 차지함. 문단 태그는 사용하는 부분은 컨텐츠의 문단에 주로 사용이 되며, 한줄 글자를 넣을 때도 다양하게 사용됨.

#ul, ol, dl 태그 : 리스트 태그, 자식요소를 포함한다. 블록요소로 p 태그를 하위에 작성해야한다.


- ul : 순서 번호가 없는 리스트
- ol : 순서 번호가 있는 리스트
- dl : 사용자 정의 리스트

#경로
-절대경로 : 내가 저장한 파일 주소의 절대값을 기입, 파일의 저장 위치 변경시에 경로를 찾을 수 없게된다.
- ./ : 현재 위치에서
- ../ : 한 단계 위로
- / : 최상단에 위치 *웹 서버에 올렸을 떄 한하여 사용

#이미지 태그: 사용하고자 하는 이미지를 문서에 링크

- 문법<img src="./img/profileImage.jpg" alt="프로필이미지"/>

- 경로를 작성시에 상대경로 값에 유의. - alt : 스크린 리더리글 사용하는 사용자가 alt 속성값의 내용을 청각정보로 접하게 된다. 웹표준의거하여 작성시에는 이미지에 있는 모든 글자를 전부 적어주어야 한다.
